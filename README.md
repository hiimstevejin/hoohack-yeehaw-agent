# livekit-meet

Minimal boilerplate for a Next.js 15 app with TypeScript, Vitest, and a stub voice agent server.

## Scripts

- `pnpm dev` runs the Next.js app locally.
- `pnpm build` builds the production app.
- `pnpm start` starts the production server.
- `pnpm test` runs Vitest.
- `pnpm voice:server` starts the voice agent server stub.
- `pnpm voice:check` type-checks the voice agent server.

## Requirements

- Node.js 18 or newer
- pnpm 10.18.2 or newer
